## Descripción

Este proyecto presenta el diseño, simulación e implementación de una plataforma
experimental para evaluar estrategias de ecualización activa en baterías de litio.

La plataforma permite:
- reconfigurar dinámicamente el conexionado de celdas mediante una matriz de switches
<!-- Esto es un comentario --> - estimar SoC (State of Charge)
<!-- Esto es un comentario -->- ensayar algoritmos de balanceo activo
- validar resultados mediante simulación y prototipo físico

El objetivo es contar con una prueba de concepto flexible para comparar
distintas arquitecturas de balanceo y analizar su eficiencia energética.
